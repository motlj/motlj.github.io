Final Project | Around the World

My final project at devCodeCamp was to create an application using a MEAN stack. I chose to create an app that grabbed a users location and used that to access a mulititude of APIs to return data. Even more than that, The app allows a user to see all of this same data for anywhere on the planet, but more specifically, on the exact opposite side of the earth.

In order to retrieve so much data, several APIs were used including, Google, Twitter, and Yahoo. Learning how to pass specific data returned from one API into a different API call was something that I found to be quite challenging, but also quite rewarding when successful. I hope to continue expanding on this application and incorporating even more APIs to further the user's experience.
