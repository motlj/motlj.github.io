kodeKiwi | A collaborative coding Site

My second big programming assignment was to contribute in a group to create a collaborative coding website. This site would allow a user to select a file from any public repository on Github, make changes to it, and if they are a contributor of that repo, push it back to Github.

The coolest feature of kodeKiwi was that multiple users could be working on the same file and watching eachother code live. We blocked out several lines of code around where one user was editting to avoid any conflicts. We also implemented a chat feature for all users viewing a specific file so that communication between teammates could happen more seamlessly.
