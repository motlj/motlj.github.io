J. Marie Sign & Design An E-Commerce Site.

J. Marie Sign & Design is an e-commerce site, through and through. Built on a LAMP stack with PHPMyAdmin, the site boasts a branded flavor on the front end and object oriented CRUD on the backend. This site was built with user stories in mind, to give customers a great user experience. Additionally, the project entailed spinning up and configuring a VM on Amazon Web Services to host the site.

kodeKiwi A collaborative coding site

The goal of kodeKiwi is to bring developers closer together when collaborating. To streamline the process of collaboration, kodeKiwi integrates with Github’s workflow and makes use of web sockets for live group work. The project aims to give a team of developers the sensation that they are part of a single unit by allowing multiple members to work on the same file at the same time and view each other’s changes live. Members also have the ability to chat within the web-based client, while they code, bringing teams of remote developers closer than ever. This project was a team effort, and built with MEAN stack.

Around the World A MEAN Stack Application.

Around the World was an idea I came up with about half way through the devCodeCamp program. It was far and away the most challenging project during my time there. This app uses many different APIs to show the user location related data. The application also allows the user to view that same data, but for the city on the exact opposite side of the planet.
